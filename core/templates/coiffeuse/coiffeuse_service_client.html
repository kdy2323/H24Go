❓ Questions fréquentes

Utilisez une carte Visa ou Mastercard. Le paiement Bancontact peut parfois échouer. Si le problème persiste, contactez notre support.

Depuis votre dashboard, cliquez sur "📅 Gérer mes horaires" pour définir vos horaires jour par jour. Vous pouvez aussi activer/désactiver votre disponibilité immédiate avec le bouton "Disponible / Indisponible".

Allez dans "Prestations" pour modifier vos services (coiffure, manucure, pose de cils, maquillage) et ajouter vos liens Instagram, TikTok ou site web.

Dans "Mon Salon", vous pouvez modifier le nom du salon, l'adresse, la ville et le numéro de téléphone visible par les clientes.

Dans "Prestations", cochez "Je veux créer mon site web professionnel". Notre équipe recevra votre demande et vous contactera pour en discuter.

Vérifiez que :

✔ Votre compte est bien actif (inscription payée)
✔ Votre bouton de disponibilité est sur "Disponible"
✔ Votre ville est correctement renseignée dans "Mon Salon"
✔ Vos services sont bien cochés dans "Prestations"
